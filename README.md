## How to Install XWiki on CentOS 7

#### 1. Install Java
```sh
sudo yum install java-1.8.0-openjdk
```
```sh
sudo java --version
```

```sh
openjdk version "1.8.0_181"
OpenJDK Runtime Environment (build 1.8.0_181-b13)
OpenJDK 64-Bit Server VM (build 25.181-b13, mixed mode)
```
